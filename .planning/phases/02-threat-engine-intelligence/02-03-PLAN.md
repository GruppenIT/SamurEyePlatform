**Create `server/services/scoringEngine.ts`:**

Class `ScoringEngineService` (singleton pattern, `createLogger('scoringEngine')`):

1. **`computeContextualScore(threat, host, journeyType, edrStatus)`** — pure function, returns ScoreBreakdownRecord:
   - `baseSeverityWeight`: critical=100, high=75, medium=50, low=25
   - `criticalityMultiplier`: domain=1.5, server/firewall/router=1.2, desktop/switch/other=1.0 (from host?.type, default 1.0 if no host)
   - `exposureFactor`: attack_surface=1.3, ad_security=1.0, edr_av=0.9, web_application=1.2
   - `controlsReductionFactor`: edrStatus='passed'→0.85, else→1.0
   - `exploitabilityMultiplier`: 1.3 if threat type is 'nmap_vuln' OR evidence has nuclei match confirmation, else 1.0
   - `rawScore`: baseSeverity * (criticalityMult * 0.25 + exposureFact * 0.20 + controlsFact * 0.15) + baseSeverity * 0.40. Then multiply by exploitabilityMultiplier.
   - `normalizedScore`: clamp rawScore to 0-100 range: `Math.max(0, Math.min(100, rawScore))`

2. **`scoreAllThreatsForJob(jobId: string)`** — queries all threats for job (both parents and children), for each:
   - Get host from hostId
   - Determine journey type from job
   - Look up EDR status: query threats table for `category='edr_av' AND status='closed' AND correlationKey LIKE 'edr:{hostname}:%'` — if found, edrStatus='passed', else 'unknown'
   - Call computeContextualScore()
   - Persist via `updateThreat(threat.id, { contextualScore: breakdown.normalizedScore, scoreBreakdown: breakdown })`

3. **`computePostureFromThreats(threats: Threat[])`** — computes overall posture score:
   - If no open threats: return 100
   - Sum all contextualScores of open threats
   - `posture = 100 - (sum / (threats.length * 100) * 100)`
   - Clamp to 0-100
   - Return score

4. **`computeProjectedScores(jobId: string)`** — for each parent threat with children:
   - Current posture = computePostureFromThreats(allOpenThreats)
   - For each parent: projected = computePostureFromThreats(allOpenThreats minus this parent's children)
   - projectedScoreAfterFix = projected - current (positive = improvement)
   - Persist via updateThreat

5. **`writePostureSnapshot(jobId: string, journeyId: string)`** — count threats by severity, compute posture, write to posture_snapshots

Export singleton: `export const scoringEngine = new ScoringEngineService();`

**In `server/routes/dashboard.ts`** — add new endpoint after existing routes:
```typescript
import { scoringEngine } from '../services/scoringEngine';
import * as postureStorage from '../storage/posture';

// POST /api/posture/simulate
app.post('/api/posture/simulate', isAuthenticatedWithPasswordCheck, async (req, res) => {
  try {
    const { threatIds } = req.body;
    if (!Array.isArray(threatIds) || threatIds.length === 0) {
      return res.status(400).json({ error: 'threatIds must be a non-empty array' });
    }
    const allOpenThreats = await getThreats({ status: 'open' });
    const currentScore = scoringEngine.computePostureFromThreats(allOpenThreats);
    const remainingThreats = allOpenThreats.filter(t => !threatIds.includes(t.id));
    const projectedScore = scoringEngine.computePostureFromThreats(remainingThreats);
    res.json({
      currentScore: Math.round(currentScore * 10) / 10,
      projectedScore: Math.round(projectedScore * 10) / 10,
      delta: Math.round((projectedScore - currentScore) * 10) / 10,
      threatsRemoved: threatIds.length,
    });
  } catch (error) {
    log.error({ err: error }, 'posture simulation failed');
    res.status(500).json({ error: 'Simulation failed' });
  }
});

// GET /api/posture/history
app.get('/api/posture/history', isAuthenticatedWithPasswordCheck, async (req, res) => {
  try {
    const { journeyId, limit } = req.query;
    const history = await postureStorage.getPostureHistory(
      journeyId as string | undefined,
      limit ? parseInt(limit as string, 10) : 30
    );
    res.json(history);
  } catch (error) {
    log.error({ err: error }, 'posture history fetch failed');
    res.status(500).json({ error: 'Failed to fetch posture history' });
  }
});
```
